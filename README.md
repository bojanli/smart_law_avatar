# 🎯 智能法律顾问数字人

> 融合大语言模型与实时3D渲染的虚拟法律顾问

## 项目简介

这是一个毕业设计项目，旨在创建一个能够进行7x24小时交互的虚拟法律顾问，通过逼真的虚拟人形象与用户自然交流。

## 技术栈

- **后端**: FastAPI + Qwen/Llama + LangChain + RAG
- **前端**: 虚幻引擎5.1+ + MetaHuman
- **语音**: Azure Speech Services
